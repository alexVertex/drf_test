<h1>Отзывы об автомобилях:</h1>

Серверная часть проекта, реазизующая базовый функционал архитектуры REST. <br/>

<br></br>
<h2>Технологии:</h2>

Python 3.12,
Django 5.0.2,
django_rest_framework 3.14.0,
djoser 2.2.2,
mysql 8.3
<br></br>
<h2>Начало работы:</h2>

Для запуска локального сервера введите в терминале команды:
`cd ElRosSite`
<br></br>
`python manage.py runserver`
